# 1 Markdown 基础

Markdown 是一种简单易学的标记语言。主要分为两部分内容：字级标记和块级标记。两者的区别是字级标记文字可以放在一行以内，而块级标记文字是不可能放到一行之内的。

{: .note }
使用 Markdown 标记过程中，所有的标记符号都要使用英文半角状态下的字母符号。使用其他符号可能导致标记失效。

## 字级标记

### 突出文字显示

这种标记用来标记单个字词。例如想突出显示“重要”这两个字，可以这样标记：

这句话很*重要*！
{: .code-example}

实现代码：

~~~ markdown
这句话很*重要*！
~~~

{: .warning}
上面这种利用斜体突出显示文字的方法在中文中并不存在，只是西文的一种表达方式。其实，我们可以将其改为文字下面加点的方式显示。

另外，还可以通过加粗文字显示其重要性：

这句话很**重要**！！
{: .code-example}

实现代码：

~~~ markdown
这句话很**重要**！！
~~~

如果，想表示顶级重要，可以叠加使用：

这句话很***重要***！！！
{: .code-example}

实现代码：

~~~ markdown
这句话很***重要***！！！
~~~

### 链接和图像

#### 链接

链接通常由链接显示文字和链接地址两部分构成。标记方法是将链接文字放入方括号中，紧跟其后就在圆括号中放入链接地址，一个链接就标记完成。例如：

[中国国家图书馆](https://nlc.cn)拥有丰富的藏书。
{: .code-example}

实现方法：

~~~ markdown
[中国国家图书馆](https://nlc.cn)拥有丰富的藏书。
~~~

可以增加链接的提示信息：

[中国国家图书馆](https://nlc.cn "中国国家图书馆官方站点")拥有丰富的藏书。
{: .code-example}

实现方法：

~~~ markdown
[中国国家图书馆](https://nlc.cn "中国国家图书馆官方站点")拥有丰富的藏书。
~~~

当把鼠标指针悬停到链接上时就会显示提示信息。

还有一种尽量不打断文字流的链接创建方式——将链接的 URL 和 提示标题用方括号括起来的参考关键词的方式使用，而对参考关键词的定义只需要在文字最后定义即可：

[中国国家图书馆][国图]拥有丰富的藏书。
{: .code-example}

[国图]: https://nlc.cn "中国国家图书馆官方网站"

实现方式：

~~~markdown
[中国国家图书馆][国图]拥有丰富的藏书。

[国图]: https://nlc.cn "中国国家图书馆官方网站"
~~~

或者，干脆直接让链接文字和参考关键词相同：

[中国国家图书馆]拥有丰富的藏书。
{: .code-example}

[中国国家图书馆]: https://nlc.cn "中国国家图书馆官方网站"

实现方式：

~~~markdown
[中国国家图书馆]拥有丰富的藏书。

[中国国家图书馆]: https://nlc.cn "中国国家图书馆官方网站"
~~~

#### 图像

图像的使用方法跟链接很相似，只需要在链接前面加一个叹号即可：

这个图像上是![青草](https://kramdown.gettalong.org/img/image.jpg)
{: .code-example}

实现方式：

~~~markdown
这个图像上是![青草](https://kramdown.gettalong.org/img/image.jpg)
~~~

也可以加上提示标题：

这个图像上是![青草](https://kramdown.gettalong.org/img/image.jpg "青青河边草")
{: .code-example}

实现方式：

~~~markdown
这个图像上是![青草](https://kramdown.gettalong.org/img/image.jpg "青青河边草")
~~~

在图像上停留鼠标指针会出现提示。

### 行内代码

行内代码使用反引号`` ` ``包围来表示：

代码 `int a = 3` 表示 `a` 的值为 3
{: .code-example}

实现方式：

~~~markdown
代码 `int a = 3` 表示 `a` 的值为 3
~~~

如果代码中需要反引号怎么办呢？只需要至少两个反引号和空格包围就可以了：

代码标识方式是 `` `code` ``
{: .code-example}

实现方式：

~~~markdown
代码标识方式是 `` `code` ``
~~~
